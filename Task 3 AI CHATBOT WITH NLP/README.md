This project is an AI chatbot developed using Python and NLTK.

### Technologies Used
- Python
- NLTK
- Scikit-learn

### How to Run
1. Install dependencies:
   pip install nltk scikit-learn numpy

2. Download NLTK data:
   python setup.py

3. Run chatbot:
   python chatbot.py

### Sample Interaction
User: What is Python?  
Bot: Python is a popular programming language.
